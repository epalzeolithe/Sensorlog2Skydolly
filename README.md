# Sensorlog2Skydolly
Convert sensorlog app to skydolly flight recorder app, to replay a flight with plane attitude and GPS position

Demo : https://www.youtube.com/shorts/5jegEKwJ2Fk

1. install sensorlog for iOS
2. attach your iphone to your plane, horizontally or verticaly (see video)
3. record attitude and GPS during flight
4. export to your computer the generated csv file
5. convert it using the python script
6. tune the python script (altitude offset, factor to pitch & roll...)
7. use the output with skydolly and MSFS
8. Enjoy
